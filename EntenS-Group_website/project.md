## СУТЬ ПРОЕКТА

Одностраничный лендинг для управляющей компании **EntenS Group** — оператора развлекательных центров, отелей и ресторанов в России. Не брокер, не консультант — **собственник и оператор**.

**Две целевые аудитории (в приоритете):**
1. **Собственник** РЦ/парка/отеля — хочет продать, сдать в аренду или передать в управление
2. **Инвестор** — хочет войти в индустрию без операционной головной боли

**Конверсия:** форма заявки → Telegram → звонок (именно в этом порядке приоритета)

## ТЕХНИЧЕСКИЙ СТЕК

```
HTML/CSS/JS:    Vanilla (без фреймворков)
Scroll/анимации: GSAP 3 + ScrollTrigger + SplitText
Smooth scroll:  Lenis
Шрифты:         Google Fonts CDN
Иконки:         Custom SVG (24x24px, зелёные)
Форма:          fetch API → Telegram Bot API или CRM
Видео:          AV1/WebM + H.264/MP4 fallback, < 2.5MB
```

**Lighthouse targets (mobile):** Performance ≥ 85 · Accessibility ≥ 90 · LCP < 2.5s · CLS < 0.1

## КЛЮЧЕВЫЕ АНИМАЦИИ

| Элемент | Тип | Детали |
|---|---|---|
| Hero H1 | GSAP SplitText | По словам, stagger 80ms, translateY(30px) + opacity |
| Trust bar числа | count-up | От 0 при scroll-into-view, easeOut |
| Секции | IntersectionObserver | fade-in + translateY(20px) |
| Портфель карточки | 3D tilt | mousemove, ~15 строк JS |
| Ambient orbs в hero | CSS keyframes | float, 25s и 35s loop, бесконечно |
| Sticky header | CSS + scroll event | backdrop-filter: blur(20px) после скролла |
| Portfolio desktop | GSAP ScrollTrigger | Горизонтальный scroll (scroll → horizontal) |
| Floating CTA | CSS | Появление через 3 сек или после 50% скролла |

**Всегда:** `@media (prefers-reduced-motion: reduce)` — отключить все анимации.

| < 480px | 1 колонка, hero H1 36px, portfolio → swipe carousel |

## ТОНАЛЬНОСТЬ

- Прямой язык предпринимателя, без корпоративного буллшита
- Конкретные цифры > абстрактные заявления
- "Мы операторы" — эта фраза должна быть слышна в каждой секции
- Белый фон + зелёные акценты = профессионализм + тепло (не холодная корпорация)
