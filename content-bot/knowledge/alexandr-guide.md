# Инструкция для Александра — согласование постов

## Твоя роль

Ты утверждаешь посты, которые Алексей подготовил и отправил на согласование. Ты не генерируешь посты сам — этим занимается Алексей. Твоя задача — проверить текст, назначить дату и одобрить публикацию.

## Короткая схема

```
Алексей → жмёт «📬 На согласование» → тебе приходит уведомление в Telegram
   → ты открываешь лист «✏️ Посты» → проверяешь текст → выбираешь дату
   → ставишь статус «К публикации» → бот публикует сам, без твоих дальнейших действий
```

## Где смотреть уведомления

Когда Алексей отправляет пост на согласование, бот сразу присылает тебе в Telegram сообщение с текстом поста и кнопкой «📊 Открыть таблицу». Нажми её — попадёшь прямо в таблицу.

## Что делать в листе «✏️ Посты»

Это единственный лист, где ты совершаешь действия.

1. Найди строку со статусом «📬 На согласование»
2. Прочитай текст поста в столбце «Текст поста» — можно поправить прямо в ячейке, если что-то не так
3. В столбце «Дата публикации» выбери дату из дропдауна (там только свободные слоты — пн/чт, по плану расписания)
4. В столбце «Статус» смени значение на «К публикации»

Дальше всё происходит автоматически: бот заберёт пост по расписанию и опубликует его 10:00 МСК в выбранную дату.

## Лист «📅 Календарь» — только смотреть

Этот лист **не для редактирования**. Он автоматически перезаписывается ботом из базы данных при каждой синхронизации (раз в сутки в 03:00 МСК, плюс сразу после твоего одобрения). Любые ручные правки в «Календаре» бесполезны — они потеряются при следующей пересборке.

Используй «Календарь» только чтобы посмотреть, что уже запланировано и что уже опубликовано — в нём теперь виден полный текст поста в столбце «Пост».

## Как поменять дату уже одобренного поста

Если нужно переставить пост на другую дату — меняй дату в столбце «Дата публикации» листа «✏️ Посты» (у поста со статусом «К публикации»). После очередной синхронизации расписание бота обновится. В «Календаре» это менять не нужно — там всё равно перезапишется автоматически.

## Если пост не нужен — статус «Удалить»

Поставь в столбце «Статус» значение «🗑 Удалить». Пост уберётся из очереди при следующей синхронизации.

## Важно

- Статус «К публикации» ставишь только ты — у Алексея нет такой возможности.
- Не трогай лист «Календарь» руками — он только для просмотра.
- Если статус «📬 На согласование» долго висит без действий — пост не уйдёт в публикацию, пока ты не назначишь дату и не сменишь статус.
