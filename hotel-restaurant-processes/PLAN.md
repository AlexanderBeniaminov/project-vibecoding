- [ ] ✅ Критерий: все данные собираются без ошибок

**Ключевые особенности iiko Cloud API:**
- Авторизация: `POST /api/1/access_token {"apiLogin": "..."}` — только API-ключ, без SHA1
- Нет OLAP-отчётов — данные берём через `/api/1/order/by_table` + агрегация в Python
- Отмены и списания: ❌ недоступны в Cloud API — в отчёте `⚠️ нет данных`
- Дата в запросах: `"yyyy-MM-dd HH:mm:ss.fff"`, не просто `"yyyy-MM-dd"`
