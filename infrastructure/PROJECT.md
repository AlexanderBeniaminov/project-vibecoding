## Ежедневный отчёт (alex) — работает ✅
Каждый день в **09:00 МСК** приходит сообщение в VK MAX с данными за предыдущий день:
- Выручка итого
- Кухня / Бар
- Средний чек
- Количество гостей
- Завтраки

Источник: Google Sheets «Отчет из IIKO», лист «ЕжеДневно»

## Команды ботов
- `/status` — состояние сервера, VPN, диска, RAM
- `/report` — последний отчёт из Google Sheets
- `/ping` — проверка связи
- `/help` — список команд
