# Парсер объявлений о продаже бизнеса

Автоматически собирает объявления о продаже развлекательных центров, батутных парков, семейных парков — с Авито, Альтера Инвест, BizTotal и других площадок. Результаты — в Google Таблице и Telegram.

---

## Запуск за 5 шагов

### Шаг 1. Установи зависимости

**Mac или Linux:**
```bash
bash install.sh
```

**Windows:**
Дважды кликни `install.bat`

---

### Шаг 2. Создай Telegram-бота

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя бота (например: `Мой Парсер`)
4. Получи токен вида `1234567890:ABCdef...` — скопируй его

**Узнай свой chat_id:**
1. Найди бота **@userinfobot** в Telegram
2. Напиши ему `/start`
3. Он покажет твой `id` — это и есть chat_id

---

### Шаг 3. Настрой Google Таблицу

<details>
<summary>Нажми чтобы раскрыть инструкцию (занимает ~10 минут)</summary>

1. Открой [console.cloud.google.com](https://console.cloud.google.com)
2. Создай новый проект (кнопка вверху, «Новый проект»)
3. В поиске введи **Google Sheets API** → включи его
4. В поиске введи **Google Drive API** → включи его
5. Меню слева: **IAM и администрирование → Сервисные аккаунты**
6. Нажми «Создать сервисный аккаунт»
7. Имя: `parser` → Создать и продолжить → Готово
8. Кликни на созданный аккаунт → вкладка «Ключи»
9. «Добавить ключ» → «Создать новый ключ» → JSON → Скачать
10. Переименуй скачанный файл в `credentials.json`
11. Положи `credentials.json` в папку проекта (рядом с `main.py`)
12. Создай новую [Google Таблицу](https://sheets.google.com)
13. Из адреса таблицы скопируй ID: `https://docs.google.com/spreadsheets/d/**ЭТО_ID**/edit`
14. Нажми «Настройки доступа» → «Добавить людей» → вставь email сервисного аккаунта (из credentials.json, поле `client_email`) → роль «Редактор»

</details>

---

### Шаг 4. Заполни файл `.env`

Открой файл `.env` в папке проекта (любым текстовым редактором) и заполни:

```
TELEGRAM_BOT_TOKEN=твой_токен_от_BotFather
TELEGRAM_CHAT_ID=твой_chat_id
GOOGLE_SPREADSHEET_ID=id_твоей_таблицы
```

---

### Шаг 5. Запусти парсер

**Mac или Linux:**
```bash
bash run.sh
```

**Windows:**
Дважды кликни `run.bat`

После запуска данные появятся в Google Таблице и придут уведомления в Telegram.

---

## Автозапуск каждый день в 09:00

**Mac/Linux:**
```bash
bash run.sh --schedule
```
Оставь окно терминала открытым (или настрой через cron — инструкция ниже).

**Windows:**
```
run.bat --schedule
```

### Настройка cron (Mac/Linux, чтобы не держать терминал)

```bash
crontab -e
```
Добавь строку (замени `/путь/к/папке` на реальный путь):
```
0 6 * * * cd /путь/к/папке && bash run.sh >> parser_cron.log 2>&1
```
*(09:00 МСК = 06:00 UTC)*

## Добавить Telegram-каналы для парсинга

1. Открой файл `telegram_sources.json`
2. Добавь username канала в массив `channels`:
```json
{
  "channels": ["@biznes_prodazha", "@gotoviy_biznes_rf"]
}
```
3. Для каналов с MTProto нужен `TELEGRAM_API_ID` и `TELEGRAM_API_HASH` в `.env`  
   (регистрация: [my.telegram.org/apps](https://my.telegram.org/apps))

---

## Добавить новое направление (например, отели)

Открой `keywords_config.json` и поменяй `"enabled": false` на `"enabled": true` для нужного направления:

```json
"hotels": {
  "enabled": true,   ← поменяй здесь
  "name": "Отели и гостиницы",
  ...
}
```

Или добавь своё направление по образцу существующих.

---

## Добавить/убрать стоп-слова

В `keywords_config.json`, раздел `"exclude_keywords"` — просто добавь слово в список:

```json
"exclude_keywords": [
  "франшиза",
  "моё новое стоп-слово"
]
```

## Структура файлов

```
├── main.py              — точка входа (запускай его)
├── config.py            — настройки (читает .env)
├── .env                 — твои секреты (не показывай никому!)
├── keywords_config.json — ключевые слова и фильтры
├── telegram_sources.json — список ТГ-каналов
├── listings.db          — база данных (создаётся автоматически)
├── parser.log           — лог работы парсера
├── credentials.json     — ключ Google API (положи сам)
├── scrapers/            — парсеры каждой площадки
└── utils/               — вспомогательные утилиты
```

---

## Что делать если что-то не работает

1. Смотри файл `parser.log` — там все ошибки
2. Запусти проверку конфига: `python main.py --check`
3. Авито заблокировал → удали `avito_session.json`, при следующем запуске откроется браузер для ручного прохождения капчи
