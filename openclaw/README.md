# OpenClaw — Mac-ассистент

Локальный AI-агент на Mac: файлы, браузер, документы, скрипты.

**Модель:** Claude Sonnet 4.6 через RouterAI (тот же ключ что у Telegram-бота)  
**Канал:** отдельный Telegram-бот (не путать с ботом-напоминалкой)

---

## Разделение с Telegram-ботом

| Телеграм-бот (VPS, 24/7) | OpenClaw (Mac, когда включён) |
|--------------------------|-------------------------------|
| Напоминалки              | Поиск файлов на Mac           |
| Google Calendar          | Браузерная автоматизация      |
| Заметки                  | Работа с PDF, docx            |
| Командные задачи         | Shell-скрипты                 |
| Работает пока Mac спит   | Работает когда за компьютером |

---

## Быстрый старт

```bash
# Установка всего одной командой:
bash openclaw/setup.sh
```

Что делает скрипт:
1. Устанавливает OpenClaw через npm
2. Копирует конфиг в `~/.openclaw/`
3. Добавляет переменные окружения в `~/.zshrc`
4. Выключает mac-indexer (больше не нужен)

После скрипта — вручную заполнить 2 значения в `~/.zshrc`:
- `ROUTERAI_API_KEY` — скопировать из `telegram-assistant/config.py`
- `OPENCLAW_TG_TOKEN` — новый бот от @BotFather

## Структура

```
openclaw/
├── README.md          # этот файл
├── setup.sh           # скрипт установки
├── openclaw.json      # конфиг (копируется в ~/.openclaw/)
└── knowledge/
    └── user.md        # контекст об Александре и проектах
```

---

## Создать нового Telegram-бота

1. Открыть @BotFather в Telegram
2. `/newbot` → дать имя (например "Alexander Mac Assistant")
3. Скопировать токен → вставить в `OPENCLAW_TG_TOKEN`

---

## Запуск

```bash
source ~/.zshrc        # загрузить переменные
openclaw start         # запустить агента
```

Или через macOS Companion App (menubar) с openclaw.ai.
