Частично вручную
🏨 Отель — 16 коттеджей + хостел ~100 мест
1
Первый контакт
  • Сайт (форма бронирования) авто
  • Телефон / АТС вручную
  • Мессенджеры / соцсети вручную
  • Менеджер по группам (личный тел.) вручную
↓
2
Обработка заявки
  • Проверка наличия в TravelLine (шахматка) вручную
  • Уточнение деталей у клиента вручную
  • Внесение брони при звонке вручную
↓
3
Подтверждение и оплата
  • Онлайн (сайт) → оплата авто
  • Телефон / группы → счёт + контроль оплаты вручную
↓
4
Заезд и обслуживание
  • Встреча на стойке (коттеджи / хостел) вручную
  • Шаттл (устно / вручную) вручную
  • Доп. услуги, ресторан частично
↓
5
Выезд и закрытие
  • Расчёт на кассе вручную
  • Закрытие брони в TravelLine вручную
↓
6
Сбор данных и отчётность
  • Данные из TravelLine вручную
  • Данные из касс вручную
  • Опрос сотрудников (шаттл, смены) вручную
  • Составление отчёта — 1–2 часа/день
🍽️ Ресторан — система iiko
1
Первый контакт
  • Гость приходит / резервирует стол частично
  • Заказ доставки (сайт / телефон / агрегатор) частично
↓
2
Обслуживание
  • Работа в iiko: заказы, блюда, чеки авто
↓
3
Закрытие смены
  • Выручка, количество гостей, чеки — в iiko авто
↓
4
Ежедневный сбор данных
  • Выручка, гости, средний чек из iiko вручную
  • Списания из iiko → копирование в Excel вручную
  • Отказы из iiko → копирование в Excel вручную
  • Внесение в общий Excel вручную
  • 20–30 мин/день
↓
5
Еженедельный сводный отчёт
  • Сбор данных из 7 ежедневных записей вручную
  • Расчёт оборачиваемости столов и мест вручную
  • Формирование итоговой таблицы вручную
  • 1–1,5 часа/неделю
⚠ Критические пробелы
  • Конверсия заявок — не отслеживается вообще
  • Звонки АТС не попадают в Битрикс24 → лиды теряются
  • Шаттл и персонал — только устный опрос
Топ-5 кандидатов на автоматизацию
Приоритет 1
Отчёт ресторана → iiko API → Google Sheets
Экономия: ~12–15 часов/месяц
iiko API + Google Sheets + Apps Script или Python
Разработка: 1–2 дня
Приоритет 2
Отчёт отеля → TravelLine API → Google Sheets
Экономия: 30–60 часов/месяц управляющего
TravelLine API + Google Sheets / Notion / Telegram-бот
Запрос каждую ночь автоматически
Приоритет 3
Дашборд собственника в реальном времени
Данные без ожидания отчётов управляющего
Google Looker Studio (бесплатно) над данными из п.1 и п.2
Приоритет 4
АТС → Битрикс24 — единая воронка заявок
Прозрачность воронки → рост конверсии
Манго / Билайн Бизнес / UIS + интеграция с Битрикс24
Средняя сложность
Приоритет 5
Авто-уведомления гостям (SMS / WhatsApp)
Снижение no-show, меньше ручных коммуникаций
TravelLine встроенные уведомления или вебхук → рассылки
Низкая сложность
